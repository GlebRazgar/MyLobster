What is OpenClaw?

OpenClaw, formerly known as Clawdbot (and briefly Moltbot), is an open-source AI assistant that can control your computer, automate complex tasks, and work across multiple platforms. With over 80,000 GitHub stars, it has quickly become one of the most popular open-source AI projects in the world.

Whether you've heard it called OpenClaw, Clawdbot, or Moltbot, it's the same powerful tool — the project has gone through a few names but was ultimately renamed to better reflect its open-source nature and growing community.

What Can OpenClaw Do?

OpenClaw is designed to be a general-purpose AI assistant with real computer interaction capabilities:

  • Computer control — OpenClaw can navigate your desktop, click buttons, type text, and interact with any application
  • Task automation — Automate repetitive workflows like data entry, file management, and web scraping
  • Multi-platform support — Works on macOS, Windows, and Linux
  • Natural language interface — Simply describe what you want in plain English
  • Extensible — Build custom plugins and integrations for your specific needs

Why Was Clawdbot / Moltbot Renamed to OpenClaw?

The project has had a few names on its journey. It started as Clawd (a pun on "Claude"), was briefly renamed to Moltbot (inspired by lobsters molting their shells), and then became Clawdbot before finally settling on OpenClaw. Each rename reflected the project's growth — and OpenClaw was chosen to emphasize its open-source, community-driven nature. If you're searching for Clawdbot or Moltbot, you're in the right place — OpenClaw is the same project with the same features and team behind it.
